Geeklog is a static Web site generator written in Python designed to be easy
to use and flexible.

# Features
* Create responsive static Web sites

# Requirements
* [Python 2.7](http://python.org/)
* [Jinja2](http://jinja.pocoo.org/)

# Directory Structure

## Geeklog Source
* geeklog
    * sites
    * lib
* tests

## Site Source
* content
* static
* templates

## Generated Site
* deploy
